## Running Oreonworlds Open World Simulation
* Run apps.oreonworlds.Main as Java Application

## Camera Control
* Move: W, A, S, D
* Rotate: Hold the middle mouse button while moving the mouse
* Accelerate Movespeed: Scroll mouse
