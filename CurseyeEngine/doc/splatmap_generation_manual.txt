Photoshop cs2
1. image -> mode -> rgb-color
2. double click on heightmap -> unlock
3. add two new layers
4. select color picker and chosse first height value from heightmap
5. select -> color range
6. cut selected color range and insert into second layer
7. repeat for next two height values
8. first layer -> crtl+A -> ctrl+C
9. channels -> add three alpha channels
10. top alpha -> crtl+V
11. repeat for next two layers
12. delete color channels
13. image -> mode -> change to rgb
